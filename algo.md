### 소수 (Prime number)
자기 자신보다 작은 수들을 나누어봐서, 하나라도 나누어지면 소수가 아닌 것
1 과 자신만 으로 분류 되는것
i.e) 1 과 2,  1, 3,  1, 5

### 약수 (Divisor)
% 나눠서 나머지가 0이면 약수.
여러개가 나올 수 있다.

### 최대공약수 (GCD)
약수중에서 가장 큰수 
i.e) 10, 5 ==> 5

### 최소공배수 (LCM)
i.e) 10, 5 ==> 10

### 순열에서 (Permutation)
{ 1, 2, 3 } 과 { 1, 3, 2 } , { 2, 1, 3 } 등.. 모두 다른 결과를 가져온다

### 조합은  (Combination)
순서가 상관이 없는 모임을 의미한다. 순서가 상관 없기 때문에
{ 1, 2, 3 }, { 1, 3, 2 } , { 2, 1, 3} 모두 같은것으로 취급을 한다.

### 중복 조합 (combination with repetition)
{1, 1} , { 1, 2 } , {1, 3 } , {2, 2 }, {2, 3}, {3, 3}
i.e) 조합: { 1, 2 } , { 1, 3 } , { 2, 3 }

### 중복 순열 (permutation with repetition)
{ 1, 1 } , { 1, 2}, {1, 3}, { 2, 1 } , {2, 2 }... 이런식으로 진행될 것이다.
i.e) 순열의 경우 { 1, 2 } , {1, 3} , { 2, 1} , {2, 3}, { 3, 1}, {3 , 2} 를 출력
