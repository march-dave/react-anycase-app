### Swap 알고리즘
장점 : 간지남. 한줄코딩가능 ex) if(A > B) A ^= B ^= A ^= B;
단점 : 코드를 이해하기 어렵다.

int A = 5;
int B = 23;

A = A ^ B;
B = A ^ B;
A = A ^ B;

### 소수 (Prime number)
자기 자신보다 작은 수들을 나누어봐서, 하나라도 나누어지면 소수가 아닌 것
1 과 자신만 으로 분류 되는것
i.e) 1 과 2,  1, 3,  1, 5
